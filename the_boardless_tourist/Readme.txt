Project Description:
The project 'The boardless tourist’ evaluates what a person’s interests are and then give them recommendations in their area to venues, restaurants, and historical destinations that we think they’ll be engaged by. 

It showcases the use of Python concepts like Functions, Lists, loops in creating the application. 

How to run:
Linux:
Open the file path > Open terminal > type - ‘python the_boardless_tourist.py’
